####  <ins>3. Building Steps:</ins>
1. Download or pull the Github repo.
2. Ensure you have NodeJS Downloaded.
3. Run the following commands:
	`npm i cors` `npm i express` `npm i path` `npm i body-parser`
4. Run the server
	`node server/server.js`
